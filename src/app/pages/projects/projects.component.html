Simulação do desenvolvimento de células cancerigenas

Projeto desenvolvido para simular tecidos cancerígenos com sistemas multiagentes usando JADE, modelando células como agentes autônomos. O simulador foi criado em Java (JADE - Framework para sistemas multiagentes). O frontend foi desenvolvido em Angular e a API que faz a comunicação via WebSocket entre o simulador e o frontend foi desenvolvida em NestJS.

Java
NestJS
Angular
JADE

Jogo MUD de Sword Art Online

Trata-se de é um jogo multiplayer baseado em texto, inspirado no universo de SAO, desenvolvido em JavaScript com PostgreSQL. Jogado via terminal, permite que os jogadores escolham entre diferentes classes, explorem o mundo, enfrentem inimigos e interajam em tempo real. Utiliza Node.js, readline para comandos interativos e um banco de dados persistente para armazenar o progresso.

JavaScript
PostgreSQL
